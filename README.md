# 🎱 福彩AI助手

> **纯工具型福彩信息查询小程序 | 微信小程序 + H5 | AI对话式交互**

---

## 项目简介

**福彩AI助手**是一款纯工具型福彩信息查询小程序，用户通过 AI 对话问答快速获取开奖信息、历史数据分析和玩法规则，**无需注册登录即可使用**。

产品定位为"数据查询助手"，而非"选号工具"或"预测平台"。通过自然语言对话，彩民可以像发微信一样提问，一句话直达结果，大幅降低中老年用户的信息查询门槛。

### 支持彩种

| 彩种 | 编码 | 说明 |
|------|------|------|
| 🎱 双色球 | `ssq` | 6红球+1蓝球，周二/四/日开奖 |
| 🔢 福彩3D | `d3` | 3位数字，每日开奖 |
| 🌈 七乐彩 | `qlc` | 7个基本号码+1特别号码 |
| ⭐ 快乐8 | `kl8` | 选一至选十，20个开奖号码 |

## 功能模块

### 🧠 AI问答中心

打开小程序即进入类 ChatGPT 的对话界面，支持 10 类标准问题：

- **开奖号码查询** — "最近一期双色球开奖号码"
- **号码遗漏统计** — "福彩3D号码123遗漏多少期"
- **历史走势统计** — "双色球红球01最近100期出现次数"
- **开奖公告/赛程** — "今天有哪些福彩开奖"
- **玩法规则** — "快乐8怎么玩"
- **数据对比查询** — "双色球最近10期蓝球"
- **期号相关** — "双色球2023150期什么时候开的"
- **中奖条件/奖金** — "双色球中4个红球多少钱"
- **历史期号范围** — "双色球最近30期开奖号码"
- **基础信息** — "福彩有哪些玩法"

AI 回答基于 **意图识别 + 数据检索** 架构，通过三层合规过滤确保回答准确、安全。

### 🏆 开奖查询

- 最新开奖结果（4 彩种首页展示）
- 历史开奖记录（按日期/期号筛选，支持翻页）
- 开奖日历/未来开奖安排
- 下一期开奖时间预测

### 📊 数据工具

| 工具 | 说明 |
|------|------|
| 走势图 | 折线图/柱状图，支持 30/50/100 期切换 |
| 遗漏统计 | 各号码遗漏期数、最大遗漏、平均遗漏 |
| 号码分布 | 红球/蓝球分布、和值分布、奇偶比、大小比 |
| 冷热分析 | 最近N期热号/冷号排行 |

### 📖 玩法规则

- 四个彩种完整玩法说明（投注规则、开奖规则、奖金设置）
- 各等奖级中奖条件与奖金对照表
- 兑奖指南（兑奖地点、期限、所需证件、注意事项）

### 📌 号码收藏

用户手动输入关注的号码组合并添加备注，仅**本地存储**（不上传服务器），不生成注单、不打印、不关联任何投注行为。

## 快速开始

### 环境要求

- Node.js ≥ 18
- MySQL 8.0（可选，无数据库时自动使用内存数据）
- npm 或 pnpm

### 1. 克隆项目

```bash
git clone https://github.com/5wjphhc244-rgb/lottery-ai-system.git
cd lottery-ai-system
```

### 2. 启动后端

```bash
cd backend
npm install

# 配置环境变量（可选）
# cp .env.example .env
# 编辑 .env 填写数据库、Mimo API 密钥等配置

npm run build
npm start
```

后端服务运行在 `http://localhost:3000`

### 3. 启动前端（H5）

```bash
cd frontend
npm install
npm run dev:h5
```

前端 H5 开发服务器运行在 `http://localhost:5173` 或 `http://localhost:8080`

### 4. 微信小程序

```bash
cd frontend
npm run dev:mp-weixin
# 用微信开发者工具导入 dist/build/mp-weixin
```

## 合规说明

### 绝对禁止（9项红线）

1. ❌ 售彩/代购/代兑
2. ❌ 预测/推荐号码
3. ❌ 诱导购彩
4. ❌ 收集个人信息（手机号、身份证、地理位置等）
5. ❌ 生成注单/投注方案
6. ❌ 投注计划提醒
7. ❌ 组织合买
8. ❌ 与站点利益绑定（分成/导流协议）
9. ❌ 线上选号→线下出票闭环

### AI 合规过滤（三层机制）

```
用户输入
  ↓
【第一层：意图识别拦截】→ 违规请求直接返回标准拒绝话术
  ↓
【第二层：生成结果审查】→ 关键词匹配 + 语义分析，拦截预测/诱导内容
  ↓
【第三层：免责声明】→ 每页显示"历史数据仅供娱乐参考，不构成投注建议"
```
