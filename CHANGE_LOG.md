## 二、任务1：全局CSS变量系统

### 修改文件：`frontend/src/uni.scss`

**变更类型：** 完全重写（161行修改）

**改动前：** 使用uni-app默认的变量值，配色以 `#007aff` 为主色，高饱和红色 `#D8000F`，纯白背景，高对比度灰色。

**改动后：**
- 引入低饱和、温暖、专业的全新配色体系：
  - **主色：** `#C41E3A`（替代 `#D8000F`，降低饱和度）
  - **蓝色球：** `#1E88E5`（替代 `#0066FF`，更柔和）
  - **绿色（温号）：** `#43A047`（柔和绿）
  - **背景色：** `#F8F8F8`（暖灰白，替代纯白）
  - **文字层级：** `#333333`（主文字）、`#666666`（正文）、`#9E9E9E`（辅助）
  - **分割线：** `#F0F0F0`（替代 `#EEEEEE`）
  - **警示背景：** `#FFF8E1`（温暖黄色，替代 `#FFF8F0`）
- 新增自定义项目变量：
  - `$shadow-sm: 0 1rpx 3rpx rgba(0,0,0,0.04), 0 4rpx 12rpx rgba(0,0,0,0.02)`
  - `$shadow-md: 0 2rpx 8rpx rgba(0,0,0,0.06), 0 8rpx 24rpx rgba(0,0,0,0.04)`
  - `$shadow-hover: 0 4rpx 16rpx rgba(0,0,0,0.08), 0 8rpx 32rpx rgba(0,0,0,0.04)`
  - `$border-radius: 16rpx`

**影响范围：** 所有页面通过uni.scss自动注入，无需在每个页面手动import。
